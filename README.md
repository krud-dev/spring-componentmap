## Installation
### Requirements

* Minimum supported Java version: 1.8
* Spring Boot 2.5.0+

### Maven
```xml
<dependency>
  <groupId>dev.krud</groupId>
  <artifactId>spring-componentmap</artifactId>
  <version>0.1.0</version>
</dependency>
```

### Gradle
#### Groovy DSL
```groovy
implementation 'dev.krud:spring-componentmap:0.1.0'
```
#### Kotlin DSL
```kotlin
implementation("dev.krud:spring-componentmap:0.1.0")
```
